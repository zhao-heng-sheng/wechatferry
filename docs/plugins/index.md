# 插件

我们提供了一个包专门用于形形色色的插件，你可以直接安装使用，你可以在这里找到 [API 参考](https://www.jsdocs.io/package/@wechatferryxxx/plugins)

## 安装

::: code-group
  ```bash [pnpm]
  pnpm add -D @wechatferryxxx/plugins
  ```
  ```bash [yarn]
  yarn add -D @wechatferryxxx/plugins
  ```
  ```bash [npm]
  npm install -D @wechatferryxxx/plugins
  ```
:::
